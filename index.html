task management

This project is a simple and user-friendly to-do list application, The app allows users to manage their daily tasks effectively. Key features include:

  • Add Tasks: Quickly add new tasks with ease.
  • Edit Tasks: Modify tasks to keep them up-to-date.
  • Remove Tasks: Delete tasks that are no longer needed.
  • Mark as Completed: Track progress by marking tasks as completed.
